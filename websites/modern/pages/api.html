API Endpoints

The API is a work in progress and is subject to change during development.

  • /api/stats - Global pool stats
  • /api/blocks - Global block stats
  • /api/pool_stats - Historical stats
  • /api/payments - Payment history
  • /api/worker_stats?taddr - Historical time per pool json
  • /api/miner_live_stats?taddr - Lightweight real-time stats (no history)
  • /api/worker_balances?taddr - Balance per pool json
  • /api/live_stats - Live stats
